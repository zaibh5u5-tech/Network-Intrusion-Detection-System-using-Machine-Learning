# Network-Intrusion-Detection-System-using-Machine-Learning

## Description
This project is a machine learning-based network intrusion detection system that identifies malicious activities in network traffic.

## Features
- Detects normal and attack traffic
- Uses machine learning algorithms
- Improves network security

## Technologies Used
- Python
- Machine Learning
- Pandas, NumPy, Scikit-learn

## How it Works
The system analyzes network data and classifies it as normal or malicious using trained ML models.

## Usage
Run the nids_code.py file to start the detection system.
